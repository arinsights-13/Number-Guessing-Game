**Number Guessing Game**

An interactive Python game where the device picks a secret number between 1 and 100, 
and ten attempts are given to guess it right!

**How to Play**
1. The computer picks a random number between 1 and 100
2. You have 10 attempts to guess it
3. After each guess you get a hint:
   - Way Too High / Way Too Low
   - Too High / Too Low
   - Nearly There! Go Higher / Go Lower
4. Guess it right and you win!
5. Run out of attempts and its game over!
